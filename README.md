# 💪 Fit Body Analyzer

A responsive BMI & Body Fat Calculator built using HTML, CSS, and JavaScript.

## ✨ Features

- 📊 BMI Calculator
- 💪 Estimated Body Fat Percentage
- 🏋️ Lean Body Mass Calculation
- ⚖️ Body Fat Weight Estimation
- 🎯 BMI Category Detection
- 📱 Responsive Design

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
